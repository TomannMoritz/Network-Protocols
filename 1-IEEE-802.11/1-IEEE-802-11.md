## QoS Dataframe
| Name              | Type          | Length    | Info |
|:------------------|:--------------|:----------|:-----|
| `frame_control`   | FrameControl  | -         |
| `flags`           | Flags         | -         |
| `_`               | -             | 1-bit     | *RESERVED*
| `duration`        | unsigned int  | 15-bit    |
| `receiver`        | unsigned int  | 48-bit    |
| `sender`          | unsigned int  | 48-bit    |
| `bss_id`          | unsigned int  | 48-bit    |
| `sn_l`            | unsigned int  | 4-bit     | Sequence Number (Lower bits)
| `fragment_number` | unsigned int  | 4-bit     |
| `sn_h`            | unsigned int  | 8-bit     | Sequence Number (Higher bits)
| `qos_control`     | QoS Control   | -         |
